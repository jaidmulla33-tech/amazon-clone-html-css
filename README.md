#  Amazon Landing Page Clone

A frontend clone of Amazon's landing page built using HTML5 and CSS3. This project was created to practice web development fundamentals, responsive layouts, and modern CSS styling techniques.

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Responsive Design

## ✨ Features

- Responsive Navigation Bar
- Search Bar
- Hero Section
- Product Categories Section
- Footer Section
- Amazon-inspired UI Design
- Clean and Well-Structured Code
- Mobile-Friendly Layout

## 📂 Project Structure

- index.html
- style.css
- images/

🚀 Getting Started

1. Clone the repository:
   
   https://jaidmulla33-tech.github.io/amazon-clone-html-css/

2. Open "index.html" in your browser.
